#  Example 01 - BAsic Read and Write

## Purpose
 The purpose of this program is to demonstrate basic input and output using low-level system calls in C. It reads up to 10 characters from the standart input and writes them to the standard output.

## System Calls 
- 'read()' : Reads data from a file description (here, standart input '0' ) into a buffer.
